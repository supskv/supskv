## Installation

```bash
# install dependencies
npm install

# .env
MQTT_USER=
MQTT_PASSWORD=
MQTT_PORT=
SERVER_PORT=

LINE_PUSH_ENDPOINT=
LINE_CHANNEL_ACCESS_TOKEN=

# serve
npm run dev

# production
npm run build && npm run start
```
